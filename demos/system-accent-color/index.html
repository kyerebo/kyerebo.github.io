AccentColor & AccentColorText

How the CSS system-color keywords render the accent color of a page — both the OS-provided default and per-element overrides.

AccentColor and AccentColorText are CSS system color keywords. AccentColor is the accent color of the page, and AccentColorText is a color guaranteed to be legible when painted on top of it.

When the accent-color property is left at its default (auto), these keywords resolve to the operating system's accent color — the same tint used by native checkboxes, radios, and range sliders. When accent-color is set to any explicit value, the keywords resolve to that color instead, and AccentColorText automatically flips between a light or dark contrasting color to stay readable.
